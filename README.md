# Anteproyecto 3 pisos - La Ceja

Modelo 3D conceptual para visualizar una alternativa preliminar de edificio multifamiliar de 3 pisos en un lote urbano de 393,3 m2 en La Ceja, Antioquia.

## Como verlo

Cuando este repositorio este publicado con GitHub Pages, abre el enlace publico del sitio. El archivo principal es:

```text
index.html
```

## Contenido

- `index.html`: app web con el modelo 3D interactivo.
- `vendor/three.module.js`: libreria Three.js guardada localmente.
- `vendor/OrbitControls.js`: controles de orbita para navegar el modelo.

## Nota importante

Este modelo es un anteproyecto conceptual para conversacion y visualizacion. No reemplaza:

- estudio de cabida arquitectonica,
- revision del POT/PBOT de La Ceja,
- calculo estructural,
- licencia urbanistica,
- revision NSR-10,
- ni concepto profesional de arquitecto o ingeniero matriculado.

La propuesta inicial usa apartamentos de 45 m2, 4 unidades por piso, 3 pisos habitables, sin ascensor, con parqueaderos para motos y bloque compacto posterior.
